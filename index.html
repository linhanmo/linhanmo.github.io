PROJECT 01

智能医学影像平台

核心开发 · Vue3 / QT6 / C++17 / 3D-RWKV
核心开发
  • 主导设计并开发肺部智能诊疗医学影像平台，采用 HTML5 + Vue3 + GSAP 构建响应式 Web 前端、QT6 + QML 开发高性能桌面客户端，统一前后端 API 接口与数据交互规范，实现跨 Windows / Linux 平台的一致体验，满足院内工作站与远程会诊双重场景。
  • 基于 C++17 搭建高性能微服务后端架构，深度解析 DICOM 3.0 标准医学影像数据，实现 CT、MRI、PET 等多模态影像统一加载与预处理；自研 3D 体绘制渲染引擎，支持百万级体素数据的实时旋转、缩放、剖切与透明度调节。
  • 部署轻量化后的 3D-RWKV 系列时空卷积模型，实现肺部组织、肺叶、肺段及结节的全自动像素级精准分割；创新性融合 CT 平扫 + 增强 + PET-CT 多模态影像特征，构建多尺度特征融合网络，精度速度两不误。
  • 设计并实现 AI 辅助手术路径规划系统，基于 3D 分割结果自动重建肺部血管、支气管与病灶的三维解剖结构，精准计算病灶与关键组织的空间距离；支持医生交互式调整切口位置与切除范围，自动生成最优手术路径并模拟切除效果。
  • 引入 Redis 集群缓存热点影像元数据、模型推理结果与用户会话信息，采用多级缓存策略大幅缩短大尺寸 3D 影像加载时间，稳定支持高并发影像分析操作。
  • 全面负责医疗数据安全体系建设，实现 TLS 1.3 传输加密 + AES-256 存储加密的全链路数据保护；集成差分隐私、同态加密等隐私保护算法，确保医疗敏感数据在计算过程中不泄露。
PROJECT 02

多模态联合情感投票分类系统

核心开发 · 音频-文本双模态 · 异构集成投票
核心开发
  • 主导设计音频-文本双模态异构集成情感分析方案，基于多模型加权投票架构实现离散区间化情感分类，解决单一模型泛化性不足、单模态特征表达片面的问题，支持正负向 5 级粒度的情感强度精准分级。
  • 构建覆盖卷积、循环、Transformer、混合架构、胶囊网络、线性统计 6 大类技术路线的异构基模型矩阵：文本侧部署 TextCNN、BiLSTM-Attention、BERT、TextRCNN、TextCapsule、FastText 共 6 种差异化模型；音频侧通过 1D-CNN 与声学 Transformer 提取梅尔频谱中的韵律、音调、语速等情感声学特征，形成多维互补。
  • 设计分层自适应加权投票融合机制：单模态内完成软投票聚合，再通过验证集网格搜索与梯度优化自适应学习各模态、各基模型权重系数，有效降低单一模型偏差与噪声干扰，显著提升系统鲁棒性。
  • 落地情感标签离散区间化映射方案，针对区间边界样本分类模糊的痛点引入边界敏感损失函数，分级准确率较传统连续回归方案显著提升，可支撑批量音视频内容的离线情感分析业务需求。
PROJECT 03

多模态虚假信息智能检测系统

核心开发 · 文本语义核验 + 视觉真伪鉴别 + 跨模态一致性校验
核心开发
  • 主导研发面向社交内容场景的多模态虚假信息检测系统，针对不实图文语义错位、视觉篡改、事实混淆等多样化造假手段，构建「文本语义核验 + 视觉真伪鉴别 + 跨模态一致性校验」三位一体的检测框架，实现虚假内容精准识别与风险等级划分。
  • 搭建覆盖 7 类差异化模型架构的异构特征提取矩阵：文本侧部署 RoBERTa、TextRCNN、BiLSTM-Attention 三类模型；图像侧基于 ResNet50、ViT、SRM 篡改检测网络提取视觉语义、深度伪造痕迹与像素级篡改特征；引入 CLIP 预训练模型完成跨模态语义初始对齐。
  • 设计「特征级细粒度对齐 + 决策级自适应投票」双层融合方案：特征层通过跨模态注意力机制实现文本实体与图像区域的语义映射；决策层融合三类检测维度的输出概率，通过网格搜索自适应分配权重，输出综合虚假置信度。
  • 落地虚假风险离散分级机制，划分为「真实 - 存疑 - 低风险虚假 - 高风险虚假」4 个等级；针对难样本引入难样本挖掘与对比学习策略优化分类边界。系统支持每日百万级内容批量检测，可对接内容审核平台实现自动化风险拦截。
PROJECT 04

轻量化多模态金融时序预测平台

核心开发 · Bi-Mamba-Freq · 多模态动态知识蒸馏
核心开发
  • 针对金融市场单模态信息片面、双向因果依赖难捕捉、多频率成分混叠、极端行情模型失效的核心痛点，主导研发 Bi-Mamba-Freq 多模态预测架构，以选择性状态空间模型为核心基座，融合量价时序、舆情文本、财报结构化特征，支撑涨跌方向分类与波动率预测双核心任务。
  • 设计参数共享式双向 Mamba 编码单元，零额外参数量开销且推理保持 O(L) 线性复杂度；创新提出 Δ 步长分组初始化机制，按高 / 中 / 低频注入时间尺度先验，单次扫描内实现多频率特征解耦；构建跨模态特征对齐模块，精准捕捉舆情事件与价格走势的长程联动关系。
  • 落地「低秩原生压缩 + 多模态动态知识蒸馏」轻量化方案：对输入输出投影层执行低秩分解；构建多模态教师—单模态学生蒸馏框架，通过方向对齐、小波域频率对齐、跨模态语义对齐三重损失，将跨模态前瞻知识高效迁移至纯量价轻量化学生模型。
  • 搭建多维度验证体系：模型涨跌分类 MCC 指标与 Transformer 类多模态 SOTA 模型持平，参数量大幅压缩，单样本推理延迟大幅降低，并发吞吐量大幅提升。
  • 设计高波动期、趋势反转期、跳空缺口三类极端行情压力测试场景，模型精度退化率控制在 1.7% 以内，显著优于传统剪枝、量化等事后压缩方案；内置 Δ 事件归因、双向分歧度拐点预警、多模态贡献度分解三类可解释性指标，满足金融监管模型合规审计要求。
